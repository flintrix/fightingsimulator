VERY IMPORTANT, IF YOU WANT TO USE THIS CODE MAKE SURE TO FOLLOW THE STEPS DOWN BELOW:
1. Make sure Java is installed.
2. Open a terminal in the project folder.
3. Run:    java -jar fighting-simulator.jar

or make a bat file with the script:
windows:
@echo off
java -jar fighting-simulator.jar
pause

linux:
#!/bin/bash
java -jar fighting-simulator.jar
read -p "Press enter to exit"
